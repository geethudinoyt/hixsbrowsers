# Quick Start Guide

## Run the Browser

### Method 1: Direct Python
```bash
python browser.py
```

### Method 2: Batch File (Windows)
Double-click `run_browser.bat`

## First Time Setup

1. Install dependencies (if not already installed):
   ```bash
   pip install -r requirements.txt
   ```

2. Run the browser using one of the methods above

## Basic Navigation

- **Address Bar**: Type URLs or search queries
- **Back/Forward**: Use arrow buttons or Alt+←/→
- **Reload**: Press F5 or click reload button
- **New Tab**: Click + button or press Ctrl+T
- **Close Tab**: Click X on tab or press Ctrl+W

## Key Features

- **Smart Search**: Type queries directly in address bar
- **Tab Management**: Drag to reorder tabs
- **Keyboard Shortcuts**: Full keyboard navigation
- **File Support**: Open local HTML files
- **Modern UI**: Clean, professional interface

## Menu Options

Click the ⋮ button to access:
- File operations (Open, Save)
- Edit functions (Find, Copy URL)
- View controls (Zoom, Full Screen)
- History management
- Help and About

## Troubleshooting

If the browser doesn't start:
1. Ensure Python is installed
2. Install PyQt5: `pip install PyQt5 PyQtWebEngine`
3. Run as administrator on Windows

Enjoy browsing!
